## Product Summary

CloudSync Pro is a cloud-based project management platform that helps remote
teams organise tasks, share files, and collaborate in real time. Core product
features:

1. **Task Manager** — Kanban and list views; assignees; due dates; subtasks; labels
2. **File Sync** — 50 GB per team; real-time sync; version history (30 days);
   supported formats: PDF, Office docs, images, ZIP
3. **Team Chat** — Threaded channels; DMs; @mentions; emoji reactions; search
4. **Analytics Dashboard** — Sprint velocity, task completion rates, team activity
5. **API Access** — REST API with OAuth2; webhooks for task/file events;
   SDKs for Python, Node.js, Ruby
